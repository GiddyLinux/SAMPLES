File Directories:

	Windows:

		/Documents/Xfer/Serum Presets/...

	Mac:

		/Library/Audio/Presets/Xfer Records/Serum Presets/...



Put the folders into the already existing ones like this:

“Cymatics - Titan Serum Expansion - Presets” into the existing "Presets" folder
“Cymatics - Titan Serum Expansion - Wavetables” into the existing “Tables” folder



Once you load a new instance of Serum, the folders appear in the Preset/Tables lists.
